Die Server-Hubs der TAUBE-Einheiten

Was ist ein Hub?

Ein Hub ist ein zentraler Knotenpunkt, wo Tauben:

  • Akku aufladen (Induktions-Ladepads)
  • Daten abladen (automatisch via 100Gbit)
  • Software-Updates empfangen
  • Sich mit anderen Tauben vernetzen
  • Neue Einsatzbefehle erhalten

Hub-Typen

Typ A - Dachboden-Hub (Klein)

  • Kapazität: 50 Tauben
  • Reichweite: 5km
  • Strom: Solar + Akku
  • Standort: Privatdächer, Kirchtürme

Typ B - Industriezone (Mittel)

  • Kapazität: 500 Tauben
  • Reichweite: 25km
  • Strom: Netzstrom
  • Standort: Fabriken, Kraftwerke, Silos

Typ C - Bunkerkomplex (Groß)

  • Kapazität: 5.000+ Tauben
  • Reichweite: 100km
  • Strom: Eigene Turbine + Notstrom
  • Standort: Unterirdische Bunker
  • Besonderheit: Rechenzentrum für Datenanalyse

Ladetechnik

Die Tauben werden auf speziellen Solar-Pads aufgeladen:

  • Induktionsladung: Kontaktloses Laden beim Aufsitzen
  • Schnellladung: 0-100% in 30 Minuten
  • Solar-Mesh: Pads haben eigene Solarzellen
  • Ersatz-Akku: Notfall-Akku für 24h Extra

💡 Die Tauben laden sich auch während des Fliegens über Mikro-Solarzellen im Gefieder!

Datenfluss

  1. Taube sammelt Audio/Video
  2. Taube fliegt zum nächsten Hub
  3. Daten werden verschlüsselt übertragen
  4. Hub leitet an übergeordneten Server weiter
  5. KI analysiert Inhalte
  6. Auffällige Daten werden markiert
  7. Bericht wird an Zentrale gesendet

Verschlüsselung: AES-256 Ende-zu-Ende

Bekannte Standorte (Deutschland)

  • Berlin: 14 Hubs (Typ B + C)
  • München: 8 Hubs
  • Hamburg: 7 Hubs
  • Frankfurt: 5 Hubs (inkl. Bunker)
  • Dörfer: Typ A an jedem Kirchturm

Schätzung: Über 3.000 Hubs bundesweit

Sicherheit

  • Biometrische Tauben-Erkennung
  • Auto-Delete bei "Gefahr"
  • Jammming-Schutz
  • Redundante Datenpfade
  • 24/7 Überwachung

Totmann-Prozedur

Wenn eine Taube "stirbt" (z.B. gefangen wird):

  1. Signal wird an Hub gesendet
  2. Alle Daten werden sofort gelöscht
  3. Letzte 24h werden an Sicherheits-Server gesendet
  4. Alle Tauben im Umkreis werden gewarnt
  5. Mission "Aborted" für 48h
